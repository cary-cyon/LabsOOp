#Задания на лабораторные
## Задание 1
```
Необходимо реализовать классы, которые выводят на печать текущую дату и время.
Один в европейском стиле, другой в американском.
Так же необходимо реализовать классы (декораторы), которые добавляют в строки даты и времени несколько произвольных символов (по вариантам).

В результате должно получиться так, что программист именяя только файл Programm.cs, может применять декораторы в прозвольных количествах и произвольном порядке.

Использовать CultereInfo для формата даты и времения
Использовать StringBuilder для создания результурующих строк
```
## Задание 2
```
Необходимо реализовать 3 операции над массивом целых чисел

1) Каждый элемент увеличивается на 1
2) Среднее элементов записывается в первый элемент 
3) Наименьшее значение записывается в последний элемент 

Количество и порядок операций определяется в файле Programm.cs
Каждая операция реализуется в своем классе
```
## Задание 2
```
Реализовать простейшее дерево (граф без циклов согласно математике и  просто класс, который имеет некоторое количество потомков, каждый из которых может иметь некоторое количество потомков в том, числе ни одного) 

В дереве должны быть 2 типа элементов, ветки (имеют потомков) и листья (хранят значение типа string). 

У ветки должна быть реализована функция, выводящая на экран все значения своих «детей», и детей вложенных в нее веток.

Конфигурация дерева должна определяться в файле Programm.cs.
```
