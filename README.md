#Задания на лабораторные
## Задание 1
```
Необходимо реализовать классы, которые выводят на печать текущую дату и время.
Один в европейском стиле, другой в американском.
Так же необходимо реализовать классы (декораторы), которые добавляют в строки даты и времени несколько произвольных символов (по вариантам).

В результате должно получиться так, что программист именяя только файл Programm.cs, может применять декораторы в прозвольных количествах и произвольном порядке.

Использовать CultereInfo для формата даты и времения
Использовать StringBuilder для создания результурующих строк
```
